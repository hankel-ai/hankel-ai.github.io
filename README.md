# hankel.ai

Personal technical portfolio built with [Hugo](https://gohugo.io/) and the [hugo-profile](https://github.com/gurusabarish/hugo-profile) theme.

## Local Development

```bash
hugo server --buildDrafts
```

## Build

```bash
hugo --gc --minify
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
